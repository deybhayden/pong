Add classic Pong blips using the WebAudio API (no asset files needed).

- One short square-wave tone on paddle hit (higher pitch)
- Different tone on wall hit (lower pitch)
- A longer tone on score
- Lazy-create `AudioContext` on first user input to satisfy autoplay policies
- Mute toggle bound to `M`

Files: `src/main.js` (new `audio.js` module recommended), help text in `index.html`.
