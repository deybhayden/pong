Added WebAudio sound effects.

Implementation:
- New `src/audio.js` module with lazy-created `AudioContext` (created/resumed inside the `keydown` handler to satisfy autoplay policy).
- Square-wave blips: paddle hit 480 Hz / 50 ms, wall hit 240 Hz / 50 ms, score 160 Hz / 350 ms (slightly louder).
- Tiny attack + linear decay envelope so tones don't click.
- Mute toggle on `M`; HUD mode label appends ` · MUTED` when muted.
- `paddleHit()` already returned bool; main loop now plays the paddle blip on a true return and the wall blip on each y-clamp branch.
- Help text in `index.html` updated with `M = mute`.

Verified `node --check`, prettier, and `vite build` all pass.
