Upgrade visuals from flat planes to a proper 3D scene while keeping 2D gameplay.

- Replace `PlaneGeometry` paddles/ball/walls with `BoxGeometry`
- Switch to `MeshStandardMaterial` (or `MeshPhongMaterial` for cheaper)
- Add lights: one `DirectionalLight` + soft `AmbientLight` (or `HemisphereLight`)
- Switch to `PerspectiveCamera` positioned slightly above and behind, looking at origin; recompute fit on resize so the full field stays visible
- Add a floor/back-wall plane for depth cues
- Optional: subtle bloom via `postprocessing` or `EffectComposer` for the classic glow

Files: `src/main.js` (scene/material/camera changes only — physics stays in 2D x/y).
