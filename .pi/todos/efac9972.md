Add a simple AI for the right paddle so the game is playable solo.

- Toggle (e.g. `1` = 1P vs AI, `2` = 2P local) on the start/serve screen
- AI tracks ball.y with a capped paddle speed (slower than `PADDLE_SPEED`) so it can be beaten
- Add a small reaction deadzone / prediction error to avoid perfect play
- Only react when ball is moving toward the AI side (otherwise drift toward center)

Files: `src/main.js` (new `updateAI(dt)` called from `tick`), HUD hint in `index.html`.
